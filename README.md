Комманда запуска .\NoNPL.exe vocabSize fileformat "datasetPath"<br>
params:<br>
vocabSize - желаемый размер словаря после завершения обучения<br>
fileformat - Тип файла куда сохраняется словарь и слияния: Json, MessagePack<br>
datasetPath - путь к датасету на котором будет обучаться токенизатор<br>
